## <a name="visualize-data-with-power-bi"></a>Визуализация данных с помощью Power BI

В этом упражнении вы будете использовать Microsoft Power BI Desktop для создания модели данных и отчета, содержащего интерактивные визуализации данных.

> **Примечание**. Упражнение является частью модуля в Microsoft Learn и требует ПК с ОС Windows. Если вы являетесь слушателем курса с преподавателем, вам должны предоставить виртуальную машину под управлением Windows.

Используйте приведенную ниже ссылку, чтобы открыть упражнение в Microsoft Learn.

**[Перейти на Microsoft Learn](https://docs.microsoft.com/learn/modules/explore-fundamentals-data-visualization/5-exercise-power-bi)**
